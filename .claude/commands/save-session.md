현재 세션을 session-memories 디렉토리에 마크다운 파일로 저장하세요.

## 저장 순서

1. `session-memories/` 디렉토리가 없으면 생성하세요.

2. 현재 날짜와 시각으로 파일명을 생성하세요:
   ```
   형식: YYYY-MM-DD_HH-MM-SS.md
   예시: 2025-03-06_14-30-00.md
   ```
   실제 현재 시각은 `date '+%Y-%m-%d_%H-%M-%S'` 명령으로 가져오세요.

3. `session-memories/.config.json`이 있으면 읽어서 `retention_days` 값을 확인하고,
   그 기간이 지난 세션 파일들을 삭제하세요 (forever.md 제외).
   config 파일이 없으면 기본값 10일을 사용하세요.

4. 아래 템플릿으로 세션 파일을 작성하세요. 이 대화에서 실제로 일어난 내용을 바탕으로 채워야 합니다:

```markdown
# 세션 메모리
**날짜**: {오늘 날짜}
**시작 시각**: {세션 시작 시각 추정}
**저장 시각**: {현재 시각}

---

## ✅ 이번 세션에서 한 일
<!-- 실제로 완료된 작업들 -->

## 📋 다음에 해야 할 일
<!-- 미완료 작업, 이어서 할 것들 -->

## ⚠️ 필수 참고 사항
<!-- 에러 해결법, 중요한 결정 사항, 주의점 -->

## 🔑 다음 세션을 위한 핵심 정보
<!-- 다음 세션에서 바로 필요한 컨텍스트: 파일 경로, 변수명, API, 진행 상태 등 -->
```

5. 파일 저장 후 `session-memories/forever.md`가 있으면 그 내용도 함께 출력하여 항상 기억해야 할 항목을 상기시켜주세요.

6. 저장 완료 메시지와 파일 경로를 알려주세요.
